# Дорожная карта исследования: физика мелодии

Главный вопрос: **почему одни сочетания звуков вызывают озноб (frisson) и легко запоминаются, а другие — нет?**

MIDI даёт нам численное представление музыки (высоты, длительности, интервалы, гармония, динамика), которое можно анализировать.

## Направление 1 — Frisson (музыкальный озноб)

Гипотезы для проверки на MIDI/аудио-данных:

- **Нарушение ожиданий**: неожиданные, но осмысленные гармонические переходы (теория ожидания Хурона, IDyOM / информационная энтропия мелодии).
- **Динамические и регистровые скачки**: резкое расширение диапазона, вступление нового голоса.
- **Голосоведение**: появление высоких устойчивых нот, апподжиатуры (задержания) на сильную долю.
- **Тембр и спектр**: яркость, шероховатость (roughness), спектральный поток — анализируется по аудио, а не MIDI.

Метрики: surprise/entropy мелодии, плотность, интервальные профили, аккордовые функции.

## Направление 2 — Запоминаемость (single-listen recall)

Что делает мелодию воспроизводимой по памяти после первого прослушивания:

- **Повторяемость и мотивная структура** (motif repetition, паттерны).
- **Малый шаговый интервал** (преобладание секунд/терций, мало больших скачков).
- **Ритмическая простота и регулярность**.
- **Контурная форма** (arch / восходяще-нисходящий контур).
- **Согласованность с тональной грамматикой** (предсказуемость по тональности).

Метрики: compression ratio (насколько мелодия сжимаема — proxy для предсказуемости), n-gram повторы, энтропия интервалов, автокорреляция ритма.
